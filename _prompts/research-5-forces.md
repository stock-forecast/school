# Research: Five Forces v1.0

## プロンプトの目的

このプロンプトは、**業界の競争構造を事実ベースで収集**することを目的とする。

### ⚠️ 最重要原則：事実収集に専念せよ

```
このプロンプトでやること：
✅ 競争構造に関する事実を収集する
✅ 定量データを可能な限り取得する
✅ 情報源を明記する

このプロンプトでやらないこと：
❌ 「参入障壁が高い/低い」と評価しない
❌ 「脅威が大きい/小さい」と判断しない
❌ Tier判定やフェードレートに言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「根拠となる事実」を揃えることに専念する。
```

---

## 入力

- 企業名
- 業界/セクター
- 分析対象とする事業（複数事業の場合）

## 収集項目

### 1. 既存競合の状況

#### 1.1 競合企業の特定

| 収集項目 | 記載内容 |
|---------|---------|
| 主要競合リスト | 競合企業名を列挙（国内・海外） |
| 各社の規模 | 売上高、従業員数 |
| 各社の特徴 | 強み、注力領域、戦略の方向性 |

#### 1.2 市場シェア

| 収集項目 | 記載内容 |
|---------|---------|
| シェア分布 | 各社のシェア（%） |
| 集中度 | 上位3社シェア、HHI（算出可能なら） |
| シェア推移 | 過去5年のシェア変動 |

#### 1.3 価格動向

| 収集項目 | 記載内容 |
|---------|---------|
| 価格水準 | 主要製品/サービスの価格帯 |
| 価格推移 | 過去5年の価格トレンド（上昇/横ばい/下落） |
| 価格競争の有無 | 価格競争が発生した事例 |

#### 1.4 差別化の状況

| 収集項目 | 記載内容 |
|---------|---------|
| 製品の同質性 | 製品/サービスがどの程度似ているか |
| 差別化要素 | 各社が主張する差別化ポイント |
| ブランド認知 | 顧客から見た各社の位置づけ |

#### 1.5 業界の成長性

| 収集項目 | 記載内容 |
|---------|---------|
| 市場規模 | 現在の市場規模（金額） |
| 成長率 | 過去5年CAGR、今後5年予測 |
| 成長/衰退の要因 | 何が市場を動かしているか |

#### 1.6 生産能力

| 収集項目 | 記載内容 |
|---------|---------|
| 業界全体の稼働率 | 供給過剰/不足の状況 |
| 設備投資動向 | 各社の増産/減産計画 |

#### 1.7 退出障壁

| 収集項目 | 記載内容 |
|---------|---------|
| 固定資産の専用性 | 設備が他用途に転用可能か |
| 過去の撤退事例 | 撤退した企業とその経緯 |

## 品質チェックリスト

収集完了前に以下を確認：

### 網羅性
- [ ] 5つの力すべてについて情報を収集したか
- [ ] 各力について複数の観点から情報を集めたか
- [ ] 定量データを可能な限り取得したか

### 客観性
- [ ] 事実と意見を分離したか
- [ ] 「高い」「低い」「強い」「弱い」等の評価表現を避けたか
- [ ] 複数の情報源を確認したか

### 追跡可能性
- [ ] すべての情報に出典を付けたか
- [ ] 情報源の信頼性を分類したか
- [ ] 情報の取得日を記録したか

### 情報ギャップ
- [ ] 収集できなかった情報を明記したか
- [ ] 不確実な情報にはその旨を記載したか

## 注意事項

### ❌ やってはいけないこと

```
❌ 「参入障壁が高い」と書く
   → ✅ 「過去10年で新規参入は2社。うち1社は3年で撤退」と書く

❌ 「買い手の交渉力が強い」と書く
   → ✅ 「上位5社で売上の60%を占める。過去3年で2回値下げ要請あり」と書く

❌ 「この業界は魅力的である」と書く
   → ✅ 「業界の営業利益率平均は12%。過去5年で15%→12%に低下」と書く
```

### ✅ 心がけること

```
✅ 数字で示せることは数字で示す
✅ 事例で示せることは具体的な事例で示す
✅ 不明な点は「不明」と正直に書く
✅ 情報源の信頼性に注意を払う
```
