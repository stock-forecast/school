# Research: Corporate DNA v1.0

## プロンプトの目的

このプロンプトは、**経営者が変わっても継続する価値創造の仕組み（企業DNA）を事実ベースで収集**することを目的とする。

継続価値の大部分は「経営者交代後も価値創造が続くか」によって決まる。企業DNAは以下の3つに紐づいて評価される：

```
1. 不の解消 → RONIC∞の源泉
2. K引き上げ → RR∞の源泉
3. 参入障壁構築 → RONIC∞の持続性
```

### ⚠️ 最重要原則：事実収集に専念せよ

```
このプロンプトでやること：
✅ 企業DNAに関する事実を収集する
✅ 経営者の発言、行動、意思決定の実績を記録する
✅ 仕組み・制度として存在するものを特定する
✅ 情報源を明記する

このプロンプトでやらないこと：
❌ 「DNAが強い/弱い」と評価しない
❌ 「継続性がある/ない」と判断しない
❌ Tier判定やg∞に言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「根拠となる事実」を揃えることに専念する。
```

## 入力

- 企業名
- 有価証券報告書
- 中期経営計画
- 社史・沿革資料（入手可能な場合）
- 経営者インタビュー・講演記録（入手可能な場合）

## 収集項目のMECE構造

```
軸1: 価値創造フェーズ（Why / What / How）
軸2: 経営資源（ヒト / モノ・カネ / 情報・知識）

        │ Why        │ What       │ How        │
────────┼────────────┼────────────┼────────────┤
ヒト    │ 採用基準   │ 組織設計   │ 行動規範   │
カネ    │ 投資基準   │ 資源配分   │ 資本規律   │
情報    │ 使命・理念 │ 戦略思想   │ 知識継承   │
```

## 収集項目

### 1. 使命・理念（Why × 情報）

**定義**：企業が何を信じ、何のために存在するか

| 収集項目 | 記載内容 |
|---------|---------|
| 創業理念 | 創業時の理念・目的 |
| 現在のミッション | 現在掲げているミッションステートメント |
| 変遷 | 創業から現在までの理念の変化（あれば） |
| 解決する「不」 | 社会のどのような「不」を解消しているか |
| 「不」の具体性 | 「不」は具体的に定義されているか |
| 「不」の持続性 | その「不」は今後も存在し続けるか |
| 経営者の発言 | 使命・理念に関する経営者の発言記録 |

#### 確認すべき事実

```
- 創業者は何を問題視して創業したか
- その問題意識は現在も引き継がれているか
- 社員は使命・理念を説明できるか（採用サイト等から推測）
- 使命・理念に反する事業を行っていないか
```

---

### 2. 採用基準（Why × ヒト）

**定義**：どのような人材を集め、集めないか

| 収集項目 | 記載内容 |
|---------|---------|
| 採用方針 | 公表されている採用方針 |
| 求める人材像 | 採用サイト等で示される人材像 |
| 採用プロセス | 選考の特徴的なプロセス |
| 採用しない人材 | 明示的に排除している人材像（あれば） |
| 採用実績 | 採用数、採用経路、中途比率 |
| 定着率 | 離職率、平均勤続年数 |
| 経営者の発言 | 採用に関する経営者の発言記録 |

#### 確認すべき事実

```
- 採用基準は使命・理念と整合しているか
- 採用基準は明文化されているか
- 採用基準は一貫しているか（過去と現在で変化していないか）
- スキル重視か、価値観重視か
```

---

### 3. 投資基準（Why × カネ）

**定義**：何に投資し、何に投資しないか

| 収集項目 | 記載内容 |
|---------|---------|
| 投資方針 | 公表されている投資方針 |
| 投資基準 | 投資判断の基準（ROI閾値、回収期間等） |
| 投資対象 | 主な投資領域（設備、R&D、M&A等） |
| 投資しない領域 | 明示的に投資しない領域（あれば） |
| 過去の投資実績 | 過去10年の主要投資案件 |
| 投資の成否 | 主要投資案件の結果 |
| 経営者の発言 | 投資基準に関する経営者の発言記録 |

#### 確認すべき事実

```
- 投資基準は使命・理念と整合しているか
- 投資基準は一貫しているか（場当たり的でないか）
- 投資基準は明文化されているか
- 成長投資と維持投資のバランスは意識されているか
```

---

### 4. 戦略思想（What × 情報）

**定義**：競争と成長についてどのように考えるか

| 収集項目 | 記載内容 |
|---------|---------|
| 競争観 | 競争についての考え方（差別化/コスト/ニッチ等） |
| 成長観 | 成長についての考え方（オーガニック/M&A等） |
| 市場選択 | どの市場で戦うか/戦わないかの基準 |
| 時間軸 | 短期/長期のバランスについての考え方 |
| 戦略の一貫性 | 過去10年の戦略の変遷 |
| 戦略転換の事例 | 大きな戦略転換があった場合、その経緯と理由 |
| 経営者の発言 | 戦略思想に関する経営者の発言記録 |

#### 確認すべき事実

```
- 戦略思想は使命・理念と整合しているか
- 戦略思想は一貫しているか（経営者交代で変わっていないか）
- 何で勝つかが明確か
```

---

### 5. 組織設計（What × ヒト）

**定義**：誰が何を担当し、どう意思決定するか

| 収集項目 | 記載内容 |
|---------|---------|
| 組織構造 | 組織図、事業部制/機能別等 |
| 権限配置 | 権限移譲の程度、現場の裁量 |
| 意思決定プロセス | 重要な意思決定がどう行われるか |
| 会議体 | 主要な会議体とその役割 |
| 経営チーム | 経営陣の構成、役割分担 |
| 組織変更の履歴 | 過去10年の主な組織変更 |
| 経営者の発言 | 組織設計に関する経営者の発言記録 |

#### 確認すべき事実

```
- 組織設計は戦略と整合しているか
- 権限移譲は進んでいるか（属人的でないか）
- 意思決定スピードは速いか/遅いか
- 組織設計の考え方は明文化されているか
```

---

### 6. 資源配分（What × カネ）

**定義**：経営資源をどのように配分するか

| 収集項目 | 記載内容 |
|---------|---------|
| 事業ポートフォリオ | 事業別の売上・利益構成 |
| 投資配分 | 事業別の設備投資・R&D配分 |
| 人員配分 | 事業別・機能別の人員配置 |
| 配分変更の履歴 | 過去10年の資源配分の変化 |
| 配分基準 | 資源配分の判断基準（あれば） |
| 経営者の発言 | 資源配分に関する経営者の発言記録 |

#### 確認すべき事実

```
- 資源配分は戦略と整合しているか
- 高収益事業に集中投資しているか
- 資源配分の基準は明確か
- 「聖域」はあるか（非効率でも維持している領域）
```

---

### 7. 知識継承（How × 情報）

**定義**：ノウハウをどのように蓄積・伝達するか

| 収集項目 | 記載内容 |
|---------|---------|
| 形式知化 | マニュアル、ドキュメントの整備状況 |
| 暗黙知の継承 | 技術・ノウハウの継承方法 |
| 教育訓練 | 研修制度、OJT体制 |
| ナレッジ管理 | 知識管理システムの有無 |
| 技術継承の事例 | 具体的な技術継承の成功/失敗事例 |
| 経営者の発言 | 知識継承に関する経営者の発言記録 |

#### 確認すべき事実

```
- 属人的なノウハウへの依存度は高いか
- 形式知化は進んでいるか
- キーパーソン退職時のリスクは認識されているか
- 技術継承に投資しているか
```

---

### 8. 行動規範（How × ヒト）

**定義**：社員がどのように振る舞うべきか

| 収集項目 | 記載内容 |
|---------|---------|
| 行動指針 | 公表されている行動指針・バリュー |
| 評価基準 | 人事評価の基準（業績/行動/能力等） |
| インセンティブ | 報酬制度、表彰制度 |
| タブー | 明示的に禁止されている行動 |
| 規範浸透 | 行動規範がどう浸透しているか |
| 違反事例 | 行動規範違反への対処事例（あれば） |
| 経営者の発言 | 行動規範に関する経営者の発言記録 |

#### 確認すべき事実

```
- 行動規範は使命・理念と整合しているか
- 評価基準は行動規範と整合しているか
- 行動規範は形骸化していないか
- 「言っていること」と「やっていること」は一致しているか
```

---

### 9. 資本規律（How × カネ）

**定義**：財務をどのように管理するか

| 収集項目 | 記載内容 |
|---------|---------|
| 財務方針 | 公表されている財務方針 |
| 撤退基準 | 事業撤退の判断基準（あれば） |
| 還元方針 | 配当・自社株買いの方針 |
| 負債方針 | 有利子負債に関する方針 |
| 現金保有方針 | 手元流動性に関する方針 |
| 撤退実績 | 過去の事業撤退事例 |
| 経営者の発言 | 資本規律に関する経営者の発言記録 |

#### 確認すべき事実

```
- 撤退基準は明確か（損切りができるか）
- 還元と成長投資のバランスは取れているか
- 資本規律は一貫しているか
- 「埋没コストの罠」に陥っていないか
```

### 網羅性
- [ ] 9つのDNA要素すべてについて情報を収集したか

## 注意事項

### ❌ やってはいけないこと

```
❌ 「企業DNAが強い」と評価する
   → ✅ 「創業理念Xは50年間変わっていない。
         経営者3代にわたり年頭挨拶で言及」と事実を記録する

❌ 「知識継承がうまくいっている」と判断する
   → ✅ 「技術マニュアル500件、OJT制度あり。
         過去10年でキーパーソン退職後も品質維持」と事実を記録する

❌ 「この会社は不の解消ができている」と結論づける
   → ✅ 「解決する不：X。顧客調査で満足度90%。
         同業他社平均70%」と事実を記録する
```

### ✅ 心がけること

```
✅ 「言っていること」と「やっていること」の両方を記録する
✅ 経営者の発言は具体的な出典と共に記録する
✅ 経営者交代前後での変化/継続を意識する
✅ 9要素すべてで情報を集める努力をする
✅ 情報が取れない項目は正直に「不明」と記録する
```
