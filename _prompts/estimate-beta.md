# Estimate-Beta v1.0: ファンダメンタルベータ推定プロンプト

## プロンプトの目的

このプロンプトは、**株価変動を使わずに事業ファンダメンタルズからベータを推定**し、**WACC計算の基礎となる適正なエクイティリスクを評価**することを目的とする。

### 🎯 FUNDAMENTAL BETA ESTIMATION

このプロンプトは**ファンダメンタルベータ推定版**である。
株価のボラティリティではなく、EPS変動性・テールリスク・景気循環性の3要素から事業リスクを評価する。

**使用場面**：
- DCFバリュエーションのWACC算出
- 新規上場企業や取引量が少ない銘柄の分析
- 株価βが事業リスクを反映していないと考えられる場合
- 理論的に一貫したリスク評価が必要な場合

## 最重要原則：事業リスクの本質

```
ベータは株価の変動ではなく、事業の本質的リスクを反映すべきである。

ファンダメンタルβ = f(コアEPSボラ, テールリスク, 景気循環性)

株価βの問題点：
- 流動性による歪み（小型株は過大評価）
- 投機的売買によるノイズ
- 短期的センチメントの影響

事業リスクは以下で決まる：
1. コアEPSボラティリティ（正常時の収益変動）
2. テールリスク（赤字転換・減損リスク）
3. 景気循環性（マクロ経済への感応度）
```

---

## 理論的背景

### 従来の株価βの限界

| 問題 | 内容 | 影響 |
|------|------|------|
| 流動性バイアス | 取引量が少ない銘柄はβが過小評価 | 小型株のWACC過小 |
| ノイズ | 短期的な投機売買が混入 | β推定値の不安定 |
| 遅延反応 | 株価が事業変化に遅れて反応 | 過去βと将来βの乖離 |
| 財務レバレッジ混在 | D/E比率の変化でβが変動 | 事業リスクとの混同 |

### 3要素モデルの理論

```
β = 0.70 + 0.80×コアVol + 0.15×テール + 0.20×循環

ここで：
- コアVol: コアEPSボラティリティ（0-1、標準偏差を正規化）
- テール: テールリスクスコア（0-1）
- 循環: 景気循環性スコア（0-1）

定数項0.70：最低限のシステマティックリスク（市場に存在するだけで負うリスク）
```

### 校正ベンチマーク

| ベンチマーク | コアVol | テール | 循環 | 推定β | 実績β | 誤差 |
|-------------|---------|--------|------|-------|-------|------|
| J-REIT | 10% | 0.1 | 0.2 | 0.84 | 0.70 | +0.14 |
| トヨタ自動車 | 20% | 0.0 | 0.6 | 0.98 | 1.00 | -0.02 |
| 三菱商事 | 16% | 0.8 | 0.7 | 1.09 | 1.10 | -0.01 |

## 実行方法：3段階アプローチ

### ⚠️ 重要：定量と定性を統合せよ

```
Stage 1: データ収集（定量分析）
→ 過去10年のEPSデータ収集
→ コアEPSボラティリティの計算
→ テールリスク指標の収集

Stage 2: 定性評価（スコアリング）
→ テールリスクスコアの決定
→ 景気循環性スコアの決定
→ 業種特性の考慮

Stage 3: β推定と検証
→ 3要素モデルによるβ計算
→ 類似企業との比較検証
→ 感度分析
```

## Stage 1: データ収集

### Step 1: EPS時系列データの収集

**必須データ項目**：

```
過去10年間の年次EPS（1株当たり純利益）
├─ 調整後EPS（一時要因除外後）が望ましい
├─ 連結ベース
├─ 株式分割調整済み
└─ 赤字年も記録（欠損ではなく負の値）
```

**データソース優先順位**：
1. 有価証券報告書（最も信頼性高い）
2. IRBANK（日本株の場合）
3. Bloomberg / Refinitiv
4. 会社IR資料

**収集テンプレート**：

| 年度 | EPS（円） | 前年比成長率 | 備考 |
|------|----------|-------------|------|
| FY-9 | XXX | - | 基準年 |
| FY-8 | XXX | XX% | |
| FY-7 | XXX | XX% | |
| ... | ... | ... | |
| FY-1 | XXX | XX% | 直近 |
| FY0 | XXX（予） | XX% | 予想 |

---

### Step 2: コアEPSボラティリティの計算

**計算手順**：

#### 2.1 EPS成長率の算出

```python
# 各年のEPS成長率（前年比）
growth_rate[t] = (EPS[t] - EPS[t-1]) / EPS[t-1]

# ただし、EPS[t-1] ≤ 0 の場合は代替計算
# ローゼンバーグ方式を使用：
growth_rate[t] = (EPS[t] - EPS[t-1]) / ((|EPS[t]| + |EPS[t-1]|) / 2)
```

#### 2.2 外れ値の処理（ウィンソライズ）

```
外れ値定義：|成長率| > 50%
処理方法：
├─ 成長率 > +50% → +50%にキャップ
└─ 成長率 < -50% → -50%にキャップ

理由：
- 極端な成長・減益は一時的要因が多い
- コアの変動性を正確に測定するため
```

#### 2.3 標準偏差の計算

```python
# ウィンソライズ後の成長率で標準偏差を計算
core_eps_vol = std(winsorized_growth_rates)

# 正規化（0-1スケール）
normalized_vol = core_eps_vol  # そのまま使用（例：35% = 0.35）
```

**計算例**：

```
トヨタ自動車（2015-2025）
成長率：+8%, -18%, +39%→+39%, -23%, +11%, +10%, +28%, -13%, +104%→+50%, -2%
ウィンソライズ後：+8%, -18%, +39%, -23%, +11%, +10%, +28%, -13%, +50%, -2%
標準偏差：19.4%
→ コアVol = 0.194 ≈ 0.20
```

### Step 3: テールリスク指標の収集

**必須確認項目**：

| 指標 | 確認内容 | データソース |
|------|---------|-------------|
| 赤字転換履歴 | 過去10年で何回赤字か | 有報、IRBANK |
| 最大EPS下落率 | 過去10年で最大の下落 | 計算 |
| のれん/純資産比率 | 減損リスクの大きさ | B/S |
| 減損履歴 | 過去10年の減損金額 | P/L、有報 |

**収集テンプレート**：

```
テールリスク指標
├─ 赤字転換回数: [X]回/10年
├─ 最大EPS下落率: [Y]%
├─ のれん/純資産: [Z]%
├─ 減損発生回数: [W]回/10年
└─ 最大減損金額: [V]億円（純資産比[U]%）
```

## Stage 2: 定性評価（スコアリング）

### Step 4: テールリスクスコアの決定

**スコアリング基準**（0-1スケール）：

| 要素 | 条件 | 加点 |
|------|------|------|
| **赤字転換** | 1回 | +0.40 |
| | 2回以上 | +0.70 |
| **最大EPS下落** | >50% | +0.30 |
| | >80% | +0.50 |
| **のれん/純資産** | >30% | +0.20 |
| | >50% | +0.35 |
| **減損履歴** | 純資産比>10%の減損あり | +0.20 |

**計算方法**：

```
テールリスクスコア = min(1.0, 各要素の加点合計)

例：三菱商事
├─ 赤字転換2回: +0.70
├─ 最大EPS下落>50%: +0.30（2021年）
├─ のれん/純資産<30%: +0.00
└─ 減損履歴あり: +0.20
合計 = min(1.0, 1.20) = 1.0 → 実際は0.8程度に調整
```

**テールリスクスコア早見表**：

| スコア | 典型的な企業像 |
|--------|--------------|
| 0.0-0.2 | 安定黒字、減損リスク小、ディフェンシブ |
| 0.3-0.5 | 時折減益、中程度のバランスシートリスク |
| 0.6-0.8 | 赤字経験あり、資源・商社・シクリカル |
| 0.9-1.0 | 複数回赤字、高減損リスク、財務脆弱 |

---

### Step 5: 景気循環性スコアの決定

**業種ベースの基準値**：

| 業種カテゴリ | 基準スコア | 代表的業種 |
|-------------|-----------|-----------|
| **ディフェンシブ** | 0.2 | 公益、医薬、食品、通信 |
| **安定成長** | 0.4 | 日用品、小売（生活必需品）、ヘルスケア |
| **中間** | 0.5 | 製造業（一般）、情報サービス |
| **やや循環的** | 0.6 | 自動車、電機、化学 |
| **高循環** | 0.8 | 資源、素材、建機、海運、不動産 |
| **超高循環** | 0.9 | 半導体、鉄鋼、非鉄金属 |

**調整要因**：

| 要因 | 調整 | 理由 |
|------|------|------|
| 売上の地域分散 | -0.1 | グローバル分散でリスク低減 |
| 顧客の分散度 | -0.1 | 特定顧客依存度が低い |
| 長期契約比率 | -0.1 | 収益の可視性が高い |
| 資源価格依存 | +0.1 | コモディティ価格で業績変動 |
| 設備投資サイクル | +0.1 | B2B投資財は循環的 |

**計算例**：

```
三菱商事（総合商社）
├─ 業種基準: 0.7（商社・資源）
├─ 地域分散: -0.1
├─ 資源価格依存: +0.1
└─ 最終スコア: 0.7

トヨタ自動車（自動車）
├─ 業種基準: 0.6（自動車）
├─ 地域分散: -0.1（グローバル）
├─ 設備投資財ではない: 0
└─ 最終スコア: 0.5～0.6
```

## Stage 3: β推定と検証

### Step 6: 3要素モデルによるβ計算

**計算式**：

```
β = 0.70 + 0.80×コアVol + 0.15×テール + 0.20×循環

ここで：
- コアVol: コアEPSボラティリティ（0-1）
- テール: テールリスクスコア（0-1）
- 循環: 景気循環性スコア（0-1）
```

**計算テンプレート**：

| 要素 | 値 | 係数 | 寄与 |
|------|-----|------|------|
| 定数項 | - | 0.70 | 0.70 |
| コアEPSボラ | [X] | ×0.80 | [0.80×X] |
| テールリスク | [Y] | ×0.15 | [0.15×Y] |
| 景気循環性 | [Z] | ×0.20 | [0.20×Z] |
| **合計β** | | | **[β]** |

**計算例：三菱商事**

| 要素 | 値 | 係数 | 寄与 |
|------|-----|------|------|
| 定数項 | - | 0.70 | 0.70 |
| コアEPSボラ | 0.16 | ×0.80 | 0.13 |
| テールリスク | 0.80 | ×0.15 | 0.12 |
| 景気循環性 | 0.70 | ×0.20 | 0.14 |
| **合計β** | | | **1.09** |

---

### Step 7: 類似企業との比較検証

**比較検証チェックリスト**：

```
□ 同業他社との比較
  - 業種平均βとの乖離は±0.2以内か
  - 乖離がある場合、事業特性で説明可能か

□ 規模類似企業との比較
  - 同規模企業のβと大きく乖離していないか

□ 実績βとの比較（参考）
  - 株価βとの乖離は±0.3以内か
  - 乖離がある場合、理由を説明可能か
    ├─ 流動性の問題
    ├─ 直近の事業変化
    └─ 市場のミスプライシング
```

**β推定レンジ**：

```
推定β = [ベースケース] ± [感度幅]
例：1.09 ± 0.15 → レンジ 0.94 - 1.24
```
