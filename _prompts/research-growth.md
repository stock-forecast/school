# Research: Growth (Logistic) v1.0

## プロンプトの目的

このプロンプトは、**ロジスティック成長モデルのパラメータ（K, r, N）を事実ベースで定量化**することを目的とする。

```
ロジスティック方程式: dN/dt = rN(1 - N/K)

K = 市場容量（成長の天井）
r = 成長速度
N = 現在の規模
```

### ⚠️ 最重要原則：定量化に専念せよ

```
このプロンプトでやること：
✅ 市場容量（K）を複数の方法で推定する
✅ 成長速度（r）を実績から算出する
✅ 現在位置（N）と浸透率（N/K）を算出する
✅ Kの拡大可能性を事実から整理する

このプロンプトでやらないこと：
❌ 「成長余地が大きい/小さい」と評価しない
❌ 「Kを突破できる/できない」と判断しない
❌ Tier判定やフェードレートに言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「パラメータの定量化」に専念する。
```

## 注意事項

### ❌ やってはいけないこと

```
❌ 「成長余地は大きい」と結論づける
   → ✅ 「N/K = 25%。ロジスティック曲線上では成長期に位置」
         と位置を記述する

❌ 「Kを突破できる」と予測する
   → ✅ 「地理拡大によるK拡大効果は+300億円（公表計画ベース）。
         実現時期は2027年」と事実を記録する

❌ 「rは高い」と評価する
   → ✅ 「r = 15%。業界平均10%、類似企業レンジ8-18%」
         と比較データを示す
```

### ✅ 心がけること

```
✅ すべてのパラメータに算出根拠を明記する
✅ 複数の推定方法を使い、レンジを把握する
✅ 信頼性の低い推定は正直にその旨を記載する
✅ Industry/Company統合レポートの事実と接続する
```
