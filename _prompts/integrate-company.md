# Integrate: Company v2.0

## プロンプトの目的

このプロンプトは、**Value Chain調査、Corporate Strategy調査、企業DNA調査の結果を接続し、企業の価値創造と継続価値の全体像を整理**することを目的とする。

継続価値を決定づける3要素：
```
1. 不の解消 → RONIC∞の源泉
2. K引き上げ → RR∞の源泉
3. 参入障壁構築 → RONIC∞の持続性
```

### ⚠️ 最重要原則：関係の整理に専念せよ

```
このプロンプトでやること：
✅ Value Chainの特徴がCorporate Strategyでどう活用されているかを整理する
✅ 企業DNAが価値創造（不の解消、K引き上げ、参入障壁）にどう紐づいているかを整理する
✅ 経営者交代後も価値創造が継続するかに関する事実を整理する

このプロンプトでやらないこと：
❌ 「競争優位がある/ない」と評価しない
❌ 「継続性がある/ない」と判断しない
❌ Tier判定やg∞に言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「事実と事実の関係」を整理することに専念する。
```

### 5. 企業DNAと価値創造の接続

企業DNA調査の結果と、価値創造の3要素（不の解消、K引き上げ、参入障壁構築）の関係を整理する。

#### 5.1 不の解消との接続

「不の解消」はRONIC∞の源泉である。企業DNAの各要素が「不の解消」にどう紐づいているかを整理する。

| DNA要素 | 不の解消との関係 | 具体的事実 |
|--------|----------------|-----------|
| 使命・理念 | [解決する「不」との一致度] | [例: 創業理念「Xの不便を解消」、50年間一貫] |
| 採用基準 | [「不」を解消する人材か] | [例: 採用基準に「顧客課題への共感」を明記] |
| 投資基準 | [「不」の解消に投資しているか] | [例: R&D投資の80%が主力事業の改善] |
| 戦略思想 | [「不」の解消を軸にしているか] | [例: 「顧客の生産性向上」を戦略の中心に据える] |
| 組織設計 | [「不」の解消に適した組織か] | [例: 顧客接点部門に権限移譲] |
| 資源配分 | [「不」の解消に資源が向いているか] | [例: 主力事業に設備投資の70%を配分] |
| 知識継承 | [「不」の解消ノウハウが継承されているか] | [例: 技術マニュアル500件、OJT制度] |
| 行動規範 | [「不」の解消を促す規範か] | [例: 評価基準に「顧客貢献」を含む] |
| 資本規律 | [「不」の解消と整合した規律か] | [例: 主力事業以外は撤退基準を厳格適用] |

#### 5.2 K引き上げとの接続

「K引き上げ」はRR∞の源泉である。企業DNAの各要素が成長余地の拡大にどう紐づいているかを整理する。

| DNA要素 | K引き上げとの関係 | 具体的事実 |
|--------|-----------------|-----------|
| 使命・理念 | [市場拡張を許容する理念か] | [例: 理念が「特定市場」に限定されていない] |
| 採用基準 | [成長を担う人材を採用しているか] | [例: 新規事業経験者の中途採用を強化] |
| 投資基準 | [成長投資の基準が明確か] | [例: 新規投資はROI15%以上を基準] |
| 戦略思想 | [成長志向の戦略か] | [例: 中計で「5年で売上2倍」を掲げる] |
| 組織設計 | [成長を推進する組織があるか] | [例: 新規事業開発部門を設置] |
| 資源配分 | [成長領域に資源が配分されているか] | [例: 新規事業に投資の20%を配分] |
| 知識継承 | [成長ノウハウが蓄積されているか] | [例: 過去のM&A統合経験が形式知化] |
| 行動規範 | [挑戦を促す規範か] | [例: 「失敗を許容する」文化を明文化] |

「参入障壁構築」はRONIC∞の持続性を決める。企業DNAの各要素が参入障壁の構築・維持にどう紐づいているかを整理する。

| DNA要素 | 参入障壁との関係 | 具体的事実 |

| 戦略思想 | [障壁を意識した戦略か] | [例: 「技術で差別化」を一貫して追求] |

## 注意事項

### ❌ やってはいけないこと

```
❌ 「企業DNAが価値創造を支えている」と評価する
   → ✅ 「創業理念Xは50年間変わらず、経営者3代が
         年頭挨拶で言及。主力事業への投資比率は70%」
         と事実を記録する

❌ 「経営者交代後も継続する」と判断する
   → ✅ 「過去2回の経営者交代で、使命・理念は継続。
         戦略思想は微修正。組織設計は大幅変更」
         と事実を記録する

❌ 「不の解消ができている」と結論づける
   → ✅ 「解決する不：X。顧客調査で満足度90%。
         売上成長率は過去5年で年平均12%」
         と事実を記録する
```

### ✅ 心がけること

```
✅ 「言っていること」と「やっていること」の両方を記録する
✅ 経営者交代前後での変化/継続を具体的に記録する
✅ 仕組みと属人性を区別する
✅ 3つの調査結果間の不整合を隠さず記録する
✅ DNAの9要素と価値創造の3軸の関係を網羅的に検討する
```
