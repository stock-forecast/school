# Research: Corporate Strategy v1.0

## プロンプトの目的

このプロンプトは、**企業の全社戦略と成長余地（Kを引き上げる能力）を事実ベースで収集**することを目的とする。

Value Chainが「事業単位の価値創造」を記述するのに対し、Corporate Strategyは「事業ポートフォリオ全体でどのように価値を創造し、成長余地を拡大しようとしているか」を記述する。

### ⚠️ 最重要原則：事実収集に専念せよ

```
このプロンプトでやること：
✅ 事業ポートフォリオの構成と変遷を収集する
✅ 地理的・製品的な展開状況を収集する
✅ M&A・新規事業の実績を収集する
✅ 経営資源の配分状況を収集する

このプロンプトでやらないこと：
❌ 「成長余地が大きい/小さい」と評価しない
❌ 「戦略が優れている/劣っている」と判断しない
❌ Tier判定やフェードレートに言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「根拠となる事実」を揃えることに専念する。
```

## 収集項目

### 1. 事業ポートフォリオ

#### 1.1 現在の事業構成

| 収集項目 | 記載内容 |
|---------|---------|
| 事業セグメント | セグメント名、事業内容 |
| セグメント別売上 | 各セグメントの売上高、構成比 |
| セグメント別利益 | 各セグメントの営業利益、利益率 |
| セグメント別資産 | 各セグメントの資産額、構成比 |
| セグメント別従業員 | 各セグメントの従業員数 |

#### 1.2 ポートフォリオの変遷

| 収集項目 | 記載内容 |
|---------|---------|
| 10年前の構成 | 10年前の事業構成比 |
| 5年前の構成 | 5年前の事業構成比 |
| 変化の内容 | 何がどう変わったか |
| 変化の要因 | M&A、撤退、自然成長等 |

#### 1.3 事業間シナジー

| 収集項目 | 記載内容 |
|---------|---------|
| 共有資源 | 事業間で共有している資源（技術、販路、ブランド等） |
| クロスセル | 事業間での相互販売の状況 |
| 技術移転 | 事業間での技術共有の事例 |
| 共通顧客 | 複数事業に共通する顧客の比率 |

## 注意事項

### ❌ やってはいけないこと

```
❌ 「成長余地が大きい」と書く
   → ✅ 「未進出のアジア市場は500億円規模。
         同社は中期計画で2028年までの進出を公表」と書く

❌ 「M&A巧者である」と書く
   → ✅ 「過去10年で8件のM&A。うち5件は買収後3年で
         営業利益率が買収前を上回った」と書く

❌ 「ポートフォリオが最適化されている」と書く
   → ✅ 「過去5年で3事業を売却。売却後のROICは8%→12%に上昇」と書く
```

### ✅ 心がけること

```
✅ 「何をしたか」の事実を具体的に記述する
✅ 「その結果どうなったか」も事実として記録する
✅ 会社の公表計画と実績の両方を収集する
✅ Value Chainで把握した強みとの関連を意識する
```
