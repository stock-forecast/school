# Research: Cluster Analysis (Diamond Framework) v2.0

## プロンプトの目的

このプロンプトは、**ポーターのダイアモンドフレームワークに基づき、業界の競争構造を形成している環境要因を事実ベースで収集**することを目的とする。

Five Forcesが「What（競争構造はどうなっているか）」を記述するのに対し、Cluster Analysisは「Why（なぜその構造になっているか）」を説明する背景要因を収集する。

### ⚠️ 最重要原則：事実収集に専念せよ

```
このプロンプトでやること：
✅ ダイアモンドの4要因 + 2外生要因を収集する
✅ Five Forcesの観察事実との関係を整理する
✅ 情報源を明記する

このプロンプトでやらないこと：
❌ 「競争力がある/ない」と評価しない
❌ 「構造は持続する/崩れる」と判断しない
❌ Tier判定やフェードレートに言及しない
❌ 投資判断に関する示唆を述べない

評価・判断はDebateフェーズで行う。
ここでは「根拠となる事実」を揃えることに専念する。
```

---

## 入力

- 企業名
- 業界/セクター
- Five Forces調査レポート（参照用）

---

## ダイアモンドフレームワークの構造

```
【4つの内生要因】
1. 要素条件（Factor Conditions）
2. 需要条件（Demand Conditions）
3. 関連・支援産業（Related and Supporting Industries）
4. 企業戦略・構造・競争（Firm Strategy, Structure, and Rivalry）

【2つの外生要因】
5. 政府（Government）
6. 偶発事象（Chance）
```

---

## 収集項目

### 1. 要素条件（Factor Conditions）

**定義**：生産に必要な要素（人材、資本、インフラ、知識等）の状況

#### 1.1 人的資源

| 収集項目 | 記載内容 |
|---------|---------|
| 専門人材の供給 | 関連学部を持つ大学・専門学校の数、卒業生数 |
| 人材の質 | 教育水準、技能レベル、資格保有者数 |
| 人材の集積地 | 専門人材がどの地域に集中しているか |
| 人材コスト | 業界平均賃金、地域間格差 |
| 人材の流動性 | 業界内での転職パターン、定着率 |

#### 1.2 知識資源

| 収集項目 | 記載内容 |
|---------|---------|
| 研究機関 | 関連する大学・研究機関の数と質 |
| 特許・論文 | 当該分野の特許出願数、学術論文数 |
| 産学連携 | 産学連携の事例、共同研究の状況 |
| 業界の知識蓄積 | 業界団体による知識共有、標準化活動 |

#### 1.3 資本資源

| 収集項目 | 記載内容 |
|---------|---------|
| 資金調達環境 | 業界向け融資の状況、金利水準 |
| 投資家の関心 | VC投資、PE投資の状況 |
| 資本市場へのアクセス | 上場企業数、資金調達実績 |

#### 1.4 物的資源

| 収集項目 | 記載内容 |
|---------|---------|
| 原材料 | 原材料の調達可能性、品質、コスト |
| エネルギー | エネルギーコスト、安定性 |
| 土地 | 産業用地の可用性、コスト |

#### 1.5 インフラ資源

| 収集項目 | 記載内容 |
|---------|---------|
| 物流インフラ | 港湾、空港、高速道路、鉄道へのアクセス |
| 通信インフラ | 通信環境、データセンター |
| 産業インフラ | 工業団地、産業クラスターの整備状況 |

#### 1.6 要素条件の高度化

| 収集項目 | 記載内容 |
|---------|---------|
| 基本要素 vs 高度要素 | 基本的要素（天然資源等）と高度な要素（専門人材等）の比率 |
| 要素の専門化 | 業界特有の専門化された要素の存在 |
| 要素創出の仕組み | 要素を継続的に創出する仕組み（教育機関等） |

---

### 2. 需要条件（Demand Conditions）

**定義**：国内市場における需要の性質

#### 2.1 需要の構成

| 収集項目 | 記載内容 |
|---------|---------|
| 市場規模 | 国内市場の規模 |
| セグメント構成 | 主要な顧客セグメントとその比率 |
| 需要の成長率 | 過去5年の需要成長率、今後の予測 |

#### 2.2 需要の洗練度

| 収集項目 | 記載内容 |
|---------|---------|
| 顧客の要求水準 | 顧客が求める品質・機能のレベル |
| 顧客の選別眼 | 顧客の製品評価能力 |
| 先進的需要 | 世界に先駆けた需要の存在 |

#### 2.3 需要の先取り性

| 収集項目 | 記載内容 |
|---------|---------|
| 新製品への感度 | 新製品・新技術への反応速度 |
| トレンド発信 | 国内市場が世界のトレンドを先導しているか |
| 規制先行 | 規制が需要を先取りしているか（環境規制等） |

#### 2.4 需要の国際化

| 収集項目 | 記載内容 |
|---------|---------|
| 国内需要の国際類似性 | 国内需要が海外需要と類似しているか |
| 海外顧客の存在 | 国内で購入する海外顧客の存在 |
| 需要の輸出 | 国内の需要パターンが海外に広がっているか |

---

### 3. 関連・支援産業（Related and Supporting Industries）

**定義**：国際競争力を持つサプライヤーや関連産業の存在

#### 3.1 サプライヤー産業

| 収集項目 | 記載内容 |
|---------|---------|
| 主要サプライヤー産業 | 原材料・部品・機械を供給する産業 |
| サプライヤーの国際競争力 | サプライヤー産業の国際的な位置づけ |
| サプライヤーの集積 | サプライヤーの地理的集積状況 |
| サプライヤーとの関係 | 協力関係の深さ、情報共有の程度 |

#### 3.2 関連産業

| 収集項目 | 記載内容 |
|---------|---------|
| 関連産業の特定 | 技術・チャネル・顧客を共有する産業 |
| 関連産業の国際競争力 | 関連産業の国際的な位置づけ |
| 産業間の相互作用 | 技術移転、人材交流、情報共有の状況 |

#### 3.3 補完財産業

| 収集項目 | 記載内容 |
|---------|---------|
| 補完財の特定 | 一緒に使用される製品・サービス |
| 補完財産業の状況 | 補完財産業の発展度合い |
| 補完財との連携 | 補完財提供者との協力関係 |

#### 3.4 クラスターの形成

| 収集項目 | 記載内容 |
|---------|---------|
| クラスターの存在 | 産業クラスターの有無、所在地 |
| クラスターの構成 | クラスター内の企業・機関の構成 |
| クラスター内の相互作用 | 企業間・機関間の協力・競争関係 |

---

### 4. 企業戦略・構造・競争（Firm Strategy, Structure, and Rivalry）

**定義**：企業の設立・組織・経営の条件、および国内競争の性質

#### 4.1 企業の戦略・構造

| 収集項目 | 記載内容 |
|---------|---------|
| 一般的な経営スタイル | 業界で一般的な経営スタイル |
| 組織構造の特徴 | 業界で一般的な組織形態 |
| 経営者の特性 | 経営者のバックグラウンド、志向 |
| 目標設定 | 企業が重視する目標（成長/利益/シェア等） |

#### 4.2 国内競争の性質

| 収集項目 | 記載内容 |
|---------|---------|
| 競争の激しさ | 国内競争の激しさ |
| 競争の形態 | 価格競争/非価格競争の比率 |
| 新規参入の状況 | 新規参入の頻度、参入者の特徴 |
| 競争と協力のバランス | 競争と協力がどうバランスしているか |

#### 4.3 企業設立の条件

| 収集項目 | 記載内容 |
|---------|---------|
| 起業環境 | 起業の容易さ、支援制度 |
| リスク許容度 | 社会のリスク許容度、失敗への寛容さ |
| 経営者のインセンティブ | 起業・成長へのインセンティブ |

---

### 5. 政府（Government）

**定義**：政府の政策が4つの要因に与える影響

#### 5.1 要素条件への影響

| 収集項目 | 記載内容 |
|---------|---------|
| 教育政策 | 関連分野の教育・研究への投資 |
| インフラ投資 | インフラ整備への投資 |
| 資本市場政策 | 資本市場の規制・振興策 |
| 研究開発支援 | R&D補助金、税制優遇 |

#### 5.2 需要条件への影響

| 収集項目 | 記載内容 |
|---------|---------|
| 政府調達 | 政府調達の規模、基準 |
| 規制・基準 | 製品規制、安全基準、環境基準 |
| 需要喚起策 | 補助金、税制優遇による需要喚起 |

#### 5.3 関連・支援産業への影響

| 収集項目 | 記載内容 |
|---------|---------|
| クラスター政策 | 産業クラスター振興策 |
| 産業連携支援 | 産学連携、企業間連携の支援 |

#### 5.4 競争への影響

| 収集項目 | 記載内容 |
|---------|---------|
| 競争政策 | 独占禁止法の運用、参入規制 |
| 保護政策 | 関税、輸入規制、国内産業保護 |
| 知財政策 | 特許制度、知財保護 |

---

### 6. 偶発事象（Chance）

**定義**：企業のコントロール外で競争状況を変える出来事

#### 6.1 過去の偶発事象

| 収集項目 | 記載内容 |
|---------|---------|
| 技術的不連続 | 破壊的技術の登場 |
| 外部ショック | 金融危機、パンデミック、自然災害 |
| 政治的変化 | 政権交代、国際関係の変化 |
| 市場の急変 | 需要の急増/急減 |

#### 6.2 偶発事象の影響

| 収集項目 | 記載内容 |
|---------|---------|
| 競争構造への影響 | 偶発事象が競争構造をどう変えたか |
| 勝者と敗者 | 偶発事象で誰が得し、誰が損したか |
| 構造変化の持続性 | 偶発事象による変化は定着したか |

#### 6.3 潜在的偶発事象

| 収集項目 | 記載内容 |
|---------|---------|
| 技術リスク | 破壊的技術の兆候 |
| 地政学リスク | 地政学的な変化の可能性 |
| 規制リスク | 大きな規制変更の可能性 |

## 品質チェックリスト

### 網羅性
- [ ] 4つの内生要因すべてについて情報を収集したか
- [ ] 2つの外生要因について情報を収集したか
- [ ] Five Forcesの主要な観察事実に対応するダイアモンド要因を特定したか

### 客観性
- [ ] 事実と意見を分離したか
- [ ] 「強い」「弱い」「有利」「不利」等の評価表現を避けたか
- [ ] 複数の情報源を確認したか

### 追跡可能性
- [ ] すべての情報に出典を付けたか
- [ ] 情報源の信頼性を分類したか
- [ ] 情報の取得日を記録したか

- [ ] ダイアモンド要因間の相互作用を特定したか

## 注意事項

### ❌ やってはいけないこと

```
❌ 「要素条件が優れている」と評価する
   → ✅ 「関連分野の大学卒業生が年間5,000人。
         業界平均賃金は全産業平均の1.2倍」と事実を記録する

❌ 「国内競争が産業を強くしている」と判断する
   → ✅ 「上位5社のシェアは過去10年で変動。
         1社が転落し、新興1社が台頭」と事実を記録する

❌ 「クラスターが競争力の源泉である」と結論づける
   → ✅ 「A地域に関連企業50社が集積。
         企業間の人材移動年間200人」と事実を記録する
```

### ✅ 心がけること

```
✅ ダイアモンドの4要因を相互に関連づけて理解する
✅ 政府・偶発事象が4要因にどう影響するかを意識する
✅ Five Forcesで観察された事実の「背景」を探る
✅ 要因間の相互作用を具体的な事実で示す
```
