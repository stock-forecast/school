| 政府（車検制度） | 需要条件（定期交換需要） | 法的義務が安定需要を創出 | 買い手：需要が制度的に確保 |
| 偶発事象（原材料高騰） | 企業戦略・競争（値上げ転嫁） | 全社一斉値上げにより競争激化を回避 | 既存競合：価格競争抑制 |
| 需要条件（EC化進展） | 企業戦略・競争（取付ネットワーク構築） | EC販売には取付サービス連携が必要 | 新規参入：異業態からの参入促進 |

### 4.2 正のフィードバックループ

| ループの内容 | 関連する要因 | 現在の状態 |
|------------|------------|-----------|
| 車検制度による定期需要 → 店舗網維持 → 専門人材育成 → サービス品質維持 → 顧客信頼 → 店舗利用継続 | 政府↔需要条件↔要素条件 | 継続中（ただし人材面で弱体化兆候） |
| 国内メーカー集積 → 技術開発 → 高品質タイヤ → ブランド志向強化 → 国産需要 → メーカー投資継続 | 関連・支援産業↔需要条件 | 継続中 |
| アジアンタイヤ品質向上 → 価格競争力 → 採用拡大 → 更なる品質投資 | 関連・支援産業↔需要条件 | 継続中（シェア拡大中） |

### 4.3 負のフィードバックループ（潜在的）

| ループの内容 | 関連する要因 | 現在の状態 |
|------------|------------|-----------|
| 整備士不足 → 賃金上昇圧力 → コスト増 → 収益悪化 → 賃金抑制 → さらなる人材流出 | 要素条件↔企業戦略・競争 | 顕在化中（求人倍率5.28倍） |
| カーシェア拡大 → 個人保有減少 → タイヤ需要減 → 店舗収益悪化 → 店舗閉鎖 → 利便性低下 → さらにシェア移行 | 需要条件↔企業戦略・競争 | 潜在的（現時点ではカーシェア車両67,199台と限定的） |
| EV普及 → ガソリンスタンド減少 → 取付拠点減少 → EC対応困難化 | 偶発事象↔関連・支援産業 | 潜在的（日本のEV比率約3%と低水準） |

## 接続の信頼性

| 接続 | 信頼性 | 理由 |
|------|--------|------|
| 車検制度（政府）→ 定期交換需要（買い手） | 高 | 法的義務と需要の直接的因果関係 |
| 整備士不足（要素条件）→ 参入困難（新規参入） | 高 | 求人倍率5.28倍という定量データあり |
| 国内メーカー集積（関連・支援産業）→ 安定調達（売り手） | 高 | シェアデータと地理的近接性から明確 |
| スタッドレス需要（需要条件）→ 日本独自市場（既存競合） | 高 | 販売本数データと「世界で日本だけ」の指摘 |
| カーシェア拡大（需要条件）→ タイヤ需要減少（代替品） | 中 | 会員数増加は事実だが、タイヤ需要への影響は限定的段階 |
| EV普及（偶発事象）→ 構造変化（既存競合） | 中 | 日本のEV比率約3%と低く、影響は将来的 |
| メーカー垂直統合（関連・支援産業）→ 競争激化（既存競合） | 中 | 店舗数増加は事実だが、独立系への影響度データ不足 |
| アジアンタイヤ台頭（関連・支援産業）→ 交渉力変化（売り手） | 中 | シェア推移は明確だが、小売への交渉力変化の定量データ不足 |

**信頼性の基準**:
- 高: 因果関係を示す直接的な証拠がある
- 中: 相関関係は認められるが因果の証拠は間接的
- 低: 論理的には関連するが証拠が不足
