| SOM | カー用品店経由市場 | 約3,950億円 | SAM × カー用品店チャネル比率33.2% |

**TAMの算出**:
```
算出方法: トップダウン
計算式: 日本市場規模114億USD × 為替150円 ≒ 1.7兆円
データソース: Report Ocean（B級）、JATMA統計（A級）
```

**SAMの算出**:
```
TAMからの制約条件:
- 新車用（OEM）: 約30%（自動車メーカー直接納入）
- 市販用（リプレイスメント）: 約70%
計算: 1.7兆円 × 70% ≒ 1.19兆円
```

**SOMの算出**:
```
SAMからの制約条件:
- カー用品量販店チャネル比率: 33.2%（JMAR調査）
計算: 1.19兆円 × 33.2% ≒ 3,950億円

現在のフジシェア: 461億円 / 3,950億円 ≒ 11.7%
現実的シェア上限: 15-20%（競合との棲み分け考慮）
```

### 3.2 成長率の推移パターン

```
10年平均成長率（2015-2024）: +5.9%
5年平均成長率（2020-2024）: +6.2%（コロナ反動含む）
3年平均成長率（2022-2024）: +5.9%
直近成長率（2024年10月期）: +2.3%

パターン: 減速傾向（2022年の+10.9%から2024年の+2.3%へ）
```

### 3.3 ロジスティック曲線へのフィッティング

```
理論式: dN/dt = rN(1 - N/K)
       → 成長率 g = r(1 - N/K)

K中央値 = 700億円と仮定した場合:
- N = 461億円
- N/K = 461 / 700 = 65.9%
- 実績成長率 g = 2.3%
- 推定 r = g / (1 - N/K) = 2.3% / (1 - 0.659) = 6.7%

K下限 = 600億円の場合:
- N/K = 461 / 600 = 76.8%
- 推定 r = 2.3% / (1 - 0.768) = 9.9%

K上限 = 800億円の場合:
- N/K = 461 / 800 = 57.6%
- 推定 r = 2.3% / (1 - 0.576) = 5.4%
```
